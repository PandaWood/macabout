macabout

Mac-style system information for Linux

Quick Install

Install macabout with one command:

curl -sSL https://pandawood.github.io/macabout/install.sh | sudo bash

What is macabout?

macabout presents your Linux machine's hardware in the same format as macOS (via "About this Mac..."). Perfect for comparing specs on repurposed Mac hardware.

  • Shows CPU model, speed, and core count
  • Memory size, speed, and type
  • Graphics card with VRAM
  • Serial number
  • Distribution info with icon
